Getting Started

  1. Start the local server: Run scripts\windows\StartLocalAssistant.bat
  2. Get your API token: Check the .env file in your project folder
  3. Configure above: Enter your server URL and API token
  4. Test the connection: Click "Test Connection" to verify everything works
  5. Start writing: Visit any website and start typing in text fields
